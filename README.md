# 職場避雷針 — 公司政策透明化儀表板

> 年終 · 紅利 · 三節 · 文具 · 打卡 — 有比較才有選擇

## 功能介紹

- **7 項指標卡**：總公司數、年終門檻比例、紅利限制比例、三節平均金額、文具阻攔率、先打卡比例、平均踩雷指數
- **5 張統計圖**：
  - 各行業「需滿一年才有年終」比例（堆疊長條圖）
  - 紅利門檻分佈（甜甜圈圖）
  - 各公司三節現金 vs 全聯禮券比較（長條圖）
  - 各行業文具申請障礙指數（長條圖）
  - 各行業打卡規定嚴格程度（水平堆疊長條圖）
- **公司情報清單**：色碼標示嚴重程度 + 自動計算「踩雷指數（0–10 分）」
- **行業篩選**：支援全部 / 科技業 / 金融業 / 傳產製造 / 文創廣告 / 零售服務
- **新增情報**：Modal 表單，送出後即時更新所有圖表

## 欄位說明

| 欄位 | 說明 |
|------|------|
| 年終門檻 | 是否需滿一年才有年終獎金 |
| 紅利門檻 | 是否需滿三年（或三個計年度）才有資格領分紅 |
| 三節形式 | 現金 / 全聯禮券 / 混合 / 無 |
| 三節金額 | 全年春節＋端午＋中秋合計 |
| 文具申請障礙 | 1（正常）→ 5（幾乎不可能）障礙程度 |
| 打卡規定 | 是否要求「先打卡再回工位」規避勞基法稽查 |
| 踩雷指數 | 以上各項加總計算，0 = 無雷，10 = 極度高風險 |

## 專案結構

```
workplace-dashboard/
├── index.html   # 主頁面結構
├── style.css    # 樣式（支援深色模式）

```

## 使用方式

1. 直接在瀏覽器開啟 `index.html`（不需要任何後端或建置工具）
2. 點右上角「新增情報」填寫公司資料
3. 點行業篩選按鈕切換檢視特定行業

## 技術依賴

- [Chart.js 4.4.1](https://www.chartjs.org/) — 圖表（透過 CDN 載入）
- 純 HTML / CSS / JavaScript，無框架依賴

## 踩雷指數計算規則

| 項目 | 條件 | 加分 |
|------|------|------|
| 年終門檻 | 需滿一年 | +2 |
| 年終門檻 | 需滿一年以上 / 無替代方案 | +1 |
| 紅利門檻 | 需滿三年 | +2 |
| 紅利門檻 | 僅特定職級 | +1 |
| 三節金額 | 低於 1,000 元 | +2 |
| 三節金額 | 1,000–2,000 元 | +1 |
| 三節形式 | 完全沒有 | +2 |
| 文具障礙 | 極度阻攔（4–5） | +2 |
| 文具障礙 | 嚴格管控（3） | +1 |
| 打卡規定 | 先打卡再回工位 | +2 |
| 打卡規定 | 固定時間要求 | +1 |
